## Sistema de Propostas Imobiliárias

Projeto de portfólio desenvolvido com base em cenários reais do mercado imobiliário, com foco na prática de desenvolvimento full stack.

---

## 📌 Sobre o projeto

Aplicação simples para gestão de propostas imobiliárias, permitindo o cadastro e organização de clientes.

O objetivo é aplicar conceitos de backend, banco de dados e frontend em um sistema funcional.

---

## 🚀 Tecnologias utilizadas

### Backend
- C#
- ASP.NET Core
- Entity Framework Core

### Frontend
- Angular

### Banco de Dados
- PostgreSQL
- DBeaver

---

## 🧩 Funcionalidades atuais

✔ Cadastro de clientes  
✔ Listagem de clientes  
✔ API REST com CRUD  
✔ Integração com banco PostgreSQL  

---

## 🔄 Fluxo do sistema

1. Cliente é cadastrado  
2. Dados são armazenados no banco  
3. API disponibiliza os dados  
4. Frontend consome a API
